## 2. Route Family Rule

| Domain | Canonical route family | Do not create without impact analysis |
| --- | --- | --- |
| Source origin | `/api/admin/source-zones/*`, `/api/admin/source-origins/*` | `/api/admin/raw-material/source-zones/*` as parallel source truth |
| SKU/recipe | `/api/admin/skus/*`, `/api/admin/ingredients/*`, `/api/admin/recipes/*` | Duplicate `/master-data/skus/*` unless adapter/deprecation is documented |
| Material | `/api/admin/production/material-issues/*`, `/api/admin/production/material-receipts` | Duplicate `/api/admin/raw-material/issues/*` or nested work-order material action as primary path without impact analysis |
| Governance/override | `/api/admin/governance/*` | Hidden business-rule bypass routes outside approval/audit policy |
| Public trace | `/api/public/trace/{qrCode}` | Any admin trace response reused as public response |
