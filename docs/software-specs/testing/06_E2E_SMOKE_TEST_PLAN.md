## 7. E2E Automation Guidance

| guideline | Nội dung                                                                                                         |
| --------- | ---------------------------------------------------------------------------------------------------------------- |
| Isolation | Mỗi run dùng suffix unique hoặc cleanup/reversal hợp lệ, không sửa audit/ledger lịch sử.                         |
| Context   | Lưu entity ids theo step: source_origin_id, raw_lot_id, po_id, batch_id, qr_code, recall_case_id, sync_event_id. |
| Retry     | Chỉ retry technical flake; không retry để che domain failure.                                                    |
| Evidence  | Lưu API response, audit id, ledger id, trace snapshot id, screenshot public trace.                               |
| Browser   | Tối thiểu desktop admin + public trace anonymous; mobile/PWA là P1 nếu chưa chốt MVP.                            |
